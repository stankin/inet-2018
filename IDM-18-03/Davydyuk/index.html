Подготовка к экзамену

Что собой представляет язык JavaScript?

Язык JavaScript является расширением разметки гипертекстов HTML и по этим причинам интерпретатор языка JavaScript встроен во все браузеры и все гипертекстовые редакторы

Список:

  1. маркированный список — <ul> — каждый элемент списка <li> отмечается маркером
  2. нумерованный список — <ol> — каждый элемент списка <li> отмечается цифрой
  3. писок определений — <dl> — состоит из пар термин <dt> — </dd> определение
Название тега Использование Особенности применения
html Указывает программе просмотра страниц что это HTML документ блочный
body "Тело" документа указывает содержание видимой части документа блочный
br Перенос строки. строчный
p Новый абзац. блочный
